BEV App Overview

Build a BEV model by choosing the vehicle configuration and component fidelity. You can preview the model and parameter file, toggle environment, HVAC, and driver settings, and export the setup.

Why: This standardizes model creation, speeds up configuration, and makes selections reproducible.

  1. Pick Base Model, Design Scenario, and Vehicle Config.
  2. Choose component fidelity for each component.
  3. Set Environment, HVAC, and Driver Setup.
  4. Save Setup, Export Parameter, or Model Setup.
