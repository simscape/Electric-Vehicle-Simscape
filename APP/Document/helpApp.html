BEV App Overview

Build a BEV model by choosing the vehicle configuration and subsystem fidelity. You can preview the model and parameter file, toggle environment, HVAC, and driver settings, and export the setup.

Why: This standardizes model creation, speeds up configuration, and makes selections reproducible.

  1. Choose vehicle configuration.
  2. Load scenario file.
  3. Pick component fidelity.
  4. Set Env, HVAC, and Driver.
  5. Create Model and parameter file.
