Overview

The BatteryTableBased block provides the highest-fidelity battery representation. Electrical characteristics come from lookup tables mapping SOC and temperature to open-circuit voltage and internal resistance. The block includes a thermal port for coupling with the coolant loop and battery heater. Positive and negative relays manage HV bus connectivity.

Model: BatteryTableBased.slx

Parameters: BatteryTableBasedParams.m

Thermal Coupling: Yes (lookup-table-driven, coolant port)

Open Model

open_system('BatteryTableBased')

Ports

Inputs

  • Positive relay command - HV bus positive connection.
  • Negative relay command - HV bus negative connection.
  • Coolant port - Thermal-liquid interface for coolant and heater coupling.

Outputs

  • Voltage - Pack terminal voltage.
  • Current - Pack current.
  • SOC - State of charge.
  • Cell Voltage - Individual cell voltage.
  • Temperature - Pack temperature from thermal dynamics.

Workflows

BatteryTableBased is the default battery fidelity in the assembled BEV system model (BEVsystemModel.slx). It is used by BEVSystemModelParams.m when running range estimation (NEDC, WLTC, EPA) and battery sizing workflows.
