# Drive Unit Thermal Durability

Evaluate electric drive unit thermal behavior over extended duty cycles to assess durability margins.

| Item | Detail |
|------|--------|
| **Entry point** | `DUThermalDurability.mlx` |
| **Template** | Electro-thermal (uses `MotorDriveThermalTestbench`) |
| **Key outputs** | Temperature profiles over extended duty cycles, thermal margin assessment |

## Files

| File | Purpose |
|------|---------|
| `DUThermalDurability.mlx` | Main live script — open this to run the workflow |
| `testBenchDuraRun.m` | Durability simulation runner |
